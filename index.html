This is a quiz game that I made for a University of Utah class using HTML, CSS/Bootstrap, and Javascript!

Good luck, have fun!

This is a multiple choice quiz with only 1 right answer in each set of questions

  • You will have 90 seconds to finish
  • For every answer you get right you will receive 5 points!
  • for every answer you get wrong, 5 seconds will be taken off the timer!
  • If finish the quiz with time remaining, that will be added to your final score!
  • Good luck and have fun!
